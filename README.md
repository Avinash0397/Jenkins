### SSH Connection Manager
otssh is a custom shell utility that helps manage SSH connections efficiently. 
It supports adding, listing, updating, deleting, and connecting to remote servers using saved SSH credentials.

#### File Structure
`otssh.sh` – Main utility script

`function.sh` – Contains all function definitions
